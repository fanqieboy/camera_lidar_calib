# Changelog - LiDAR-Camera Calibration Tool

## [1.0] - 2026-04-09
### Added
- **版本定义**: 在 `main.cpp` 中通过宏定义 `MAJOR_VERSION 1` 和 `MINOR_VERSION 0` 正式确定了项目版本号，并在启动时打印。
- **核心功能**: 实现了基于 ArUco 标记和圆孔拟合的 LiDAR-Camera 外参标定流程。
- **相机检测模块**: 集成 `QRDetector`，实现 ArUco 标记识别并逆推 3D 圆心位置。
- **雷达检测模块**: 集成 `LidarDetector`，通过点云聚类和几何约束（如垂直度、圆周度）实现特征圆拟合。
- **外参解算核心**: 在 `CalibCore` 中封装了基于 3D-3D 点对（ICP-like 或 PnP-like）的 6-DOF 外参解算（变换矩阵 $T_{L->C}$）。
- **自动化流程**: `main.cpp` 中定义了从“加载配置 -> 提取特征 -> 特征匹配 -> 解算 -> 保存结果”的 7 步标定工作流。
- **可视化结果**: 支持将相机端的检测结果图以及雷达端的拟合结果（点云形式）自动保存到 `output/` 文件夹。
- **真值对比功能**: 内置与合成数据真值的误差对比逻辑，方便开发阶段进行精度验证。

### Changed
- **Config 管理**: 采用 `ConfigManager` 进行参数配置，支持在启动时通过命令行指定配置文件路径。
- **结构优化**: 采用模块化设计，将数据加载 (`DataLoader`)、检测、解算逻辑分层隔离。

### Fixed
- **数据同步**: 修复了相机与雷达特征点排序不一致导致的解算发散问题（通过 `SortPatternCenters` 函数解决）。
